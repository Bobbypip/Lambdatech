# Lambdatech_Sistema_Escuela-API_BackEnd
Sistema de registro de alumnos

El sistema deberá permitir la captura de alumnos, padres de familia y escuelas, y deberá permitir la vinculación de alumnos y padres de familia es decir un alumno puede tener un padre y una madre únicamente pero un padre y una madre pueden tener varios hijos y por otro lado el alumno puede estar vinculado a varias escuelas, mostrar un reporte de los alumnos con las escuelas a las que ha pertenecido y mostrar quienes son sus padres de familia asignados.
